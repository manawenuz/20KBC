# 20,000 BC

A prehistoric real-time strategy game set in the 20,000 BC era.

Lead your tribe through survival, resource management, and warfare in an unforgiving world of ice ages, wild predators, and rival clans.

## Game Features

- **Real-time Strategy** with survival elements
- **Multiple game modes**: Custom, Campaign, Multiplayer (Ranked/Unranked), Survival (20-30 players), Tutorial
- **Dynamic environment**: 4 seasons, 4 geographic zones, 6 climate types, day/night cycle
- **Natural disasters**: River floods, wildfires, famines, fog, wind
- **GAIA ecosystem**: Neanderthals, lions, deer, elephants, wild cows with herd AI
- **Resource economy**: Wood, stone, clay, water, food, skin, rope, supply
- **Unit progression**: Workers upgrade through 3 tiers with axes, spears, and slings
- **Buildings**: Home base (mobile/settled), farms, supply depots, walls, towers, boats
- **Tech tree**: 10 upgrades from skinning to siege ladders
