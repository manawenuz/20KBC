# 20,000 BC - Game Design Document

A prehistoric real-time strategy game set in the 20,000 BC era, where players lead their tribe through survival, resource management, and warfare in an unforgiving world.

## About This Project

- **Genre**: Real-time Strategy (RTS) with survival elements
- **Setting**: Prehistoric era (~20,000 BC)
- **Players**: 1-30+ players depending on mode
- **Status**: Design & pre-production phase

### Game Design
Core game mechanics, combat systems, and player actions.

Environment simulation: seasons, climate, GAIA ecosystem, and resources.

All unit types, buildings, and the tech tree.
