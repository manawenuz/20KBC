# Game Overview



## Genre

20,000 BC is a **real-time strategy (RTS)** game -- a subgenre of strategy video games in which the game does not progress incrementally in turns.

- **Survival mode**: A giant map for 20, 30, or even more players where the goal is to survive and destroy other players
- **Skirmish**: A small map for 1v1 or 2v2 where the main purpose is to destroy other player(s)

## Time Period

The game is set in the **20,000 BC era** -- the prehistoric period.

Prehistory is the period which has records of human activities, civilization, and the use of stone tools. This period doesn't witness any writings from the civilians, therefore it is the prehistoric period in ancient history.

Human prehistory spans from the use of the first stone tools (~3.3 million years ago) by hominins to the invention of writing systems (~5,300 years ago).

## Game Modes

### Custom Game

Players can create games and play against AI or other players.

- Players can set up various settings before starting: map size, weather, starting season, climate, season change, climate change, player locations, region's climate factor, etc.
- Players can add different AI difficulty levels
- Players can invite others through LAN or online mode

### Campaign

A storyline-based mode where the player follows different storylines across levels.

- Predefined maps with specific objectives: reach a certain population, gather a certain amount of resources, migrate from one region to another through tough terrain in bad seasons, etc.
- Supports Co-op play

### Multiplayer

Online competitive play:

- **Ranked**: 1v1 or 2v2 competitive mode where each game result impacts player rank (ELO-like system, TBD)
- **Unranked**: 1v1 or 2v2 competitive mode with no rank impact

### Survival Mode

A unique online game type with a giant map featuring ~20-30 players fighting each other to survive.

- May include a rank system (TBD)
- May merge with multiplayer mode (TBD)

### Tutorial

Single-player mode designed to teach game mechanics to new players. Similar to campaign structure.
